# 🐍 Snake Game (Console Version - C++)

A console-based Snake Game built using C++.

## Features
- Real-time movement
- Fruit generation
- Score tracking
- Collision detection
- Increasing snake length

## Controls
W - Up  
S - Down  
A - Left  
D - Right  
X - Exit  

## Concepts Used
- Game Loop
- 2D Grid Logic
- Arrays
- Enum
- Real-time Input Handling
